# Open Source Projects hosted on [Github](https://github.com/kitwalker12)

* [Brightpearl API](https://github.com/kitwalker12/brightpearl_api) - _Ruby wrapper for the Brightpearl API_
* [Spree Variant Properties](https://github.com/kitwalker12/spree_variant_properties) - _Adds Properties to Spree Variants like Products_
* [Retail Calendar](https://github.com/kitwalker12/retail_calendar) - _Gem for finding ranges in Retail or the 4-5-4 Calendar format_
* [Shopkeep Reports](https://github.com/kitwalker12/shopkeep_reports) - _Ruby wrapper for downloading Shopkeep Reports using mechanize_
* [Spree Line Item Discount](https://github.com/kitwalker12/spree_line_item_discount) - _Spree extension for adding discount to Line Items in Spree_
